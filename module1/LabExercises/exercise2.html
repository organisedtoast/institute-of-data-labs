Lab Exercise 2

Objective: Practice HTML structure and file management

My first HTML sheet

This HTML text was styled without using any CSS, as that is too hard for a beginner.

  1. Open VS Code: Open (or create) your Module1 folder in VS Code.
  2. Create Directory: Use the Terminal feature of VS Code to issue commands to create a new folder inside Module1 for LabExercises and change into it.
  3. Create File: Inside the above folder, create a new file called exercise2.html
  4. HTML Skeleton: As the first character in this file, type an exclamation mark ! and use the Emmet abbreviation to generate a basic HTML skeleton
  5. Populate Content: Populate the <body> with basic HTML representing these lab steps
  6. Preview: Go Live to view and test the HTML
  7. Format Code: Install Prettier VSCode extension and format selected code with Ctrl-K-F
  8. Additional Reading: Read through Dealing with files on the MDN for more tips about website file systems.
