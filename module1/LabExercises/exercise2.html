Objective: Practice HTML structure and file management on VS Code

My very first HTML sheet created on VS Code

This HTML text was styled without using any CSS, as that is too hard for a beginner:

  1. Open VS Code: I opened (or created) a Module1 folder in VS Code.

  2. Create Directory: I used the Terminal feature of VS Code to issue commands to create a new folder inside Module1 for LabExercises and change into it.

  3. Create File: Inside the above folder, I create a new file called exercise2.html

  4. HTML Skeleton: As the first character in this file, I typed an exclamation mark ! and used the Emmet abbreviation (tab key) to generate a basic HTML skeleton

  5. Populate Content: I populated the <body> with basic HTML representing these lab steps

  6. Augment code with Qwen Coder Companion: I instructed Qwen to make the text more visually appealing using semantic HTML elements and without resorting to any CSS styling.

  7. Preview: I went Live to view and test the HTML

  8. Format Code: I installed the Prettier VSCode extension and formatted my code base with Ctrl-K-F

  9. Additional Reading: I read through Dealing with files on Mozilla Developer Network for more tips about website file systems.
